# Modular Data Logger based on ESP32-S3

## Overview

This project was developed during the FEELT39040P course, "Special Topics in Computer Engineering: Hardware and Firmware Development for IoT," taught by Professor Marcelo Barros de Almeida, from the Computer Engineering program at the Faculty of Electrical Engineering, within the Federal University of Uberlândia.

In this course, students are encouraged to create original hardware projects focused on IoT and employing ESP32 processors.
